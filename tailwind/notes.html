Tailwind notes

Common Units

  • 0= m-0, p-0, w-0
  • px = p-[100px]
  • Numeric Scale: p-4, m-6, in terms of rem -> p-1 = p[0.25rem]
  • Keywords: w-full, h-screen, max-w-sm, min-h-full
  • Fractions: w-1/2, this is fraction of the parent or percentage as we use in css

Spacing and sizing

Padding

  • p-* = padding for all sides
  • px-* = padding for x axis i.e. left and right
  • py-* = padding for y axis i.e. top and bottom
  • pt-*, pb-*, pr-*, pl-* -> top, bottom, right, left
  • ps=*, pe-* = padding start, and padding end

Margins

  • m-* for all sides
  • my-*, m-x*
  • mt-*, mb-*, ml-*, mr-*
  • -mt-*, -mx-* : negative margin value
  • mx-auto for horizontal centering of block elements
  • Flex grid: gap-2, gap-x-4, gap-y-4
  • Spacing between sibling elements without flex and grid : space-y-2 space-x-2

Width

  • Fixed Values: w-1 w-2 w-3 w-5 w-96
  • Fractions: w-1/2 w-2/3
  • Content based- w-auto, w-fit, w-min, w-max
  • Screen based: w-screen
  • Parent relation: w-full
  • Max and Min: max-w-*, min-w-*

Height

  • Fixed: h-0, h-[100px]
  • screen and parent: h-full, h-screen
  • min and max: min-h-*, max-h-*

Typography

  • Font Sizing: text-xs, text-sm, text-base, text-lg, text-xl, text-2xl, text-6xl
  • Weight: font-thin, font-bold, font-extralight, font-[100px]
  • Font Family: font-sans, font-serif

Coloring

  • Color values-> text-gray=600 -> property-color-intensity, text-white, text-black
  • alignment: text-center, text-left, text-justify
  • Decoration: underline, line-through, no-underline
  • Transform: uppercase, lowercase, capitalize, normal-case

Background Utilities

  • color: bg-red-200
  • image: bg-[url('google.co.in/image/100x100')]
  • positioning: bg-center, bg-top, bg-bottom
  • Sizing: bg-cover, bg-contain
  • repeat: bg-no-repeat bg-repeat-x

Background Utilities

  • border, border-0, border-1, border-2..., border-b
  • border-color: border-gray-200
  • border-styles: border-solid, border-dashed, border-dotted
  • Border-radius: rounded-none, rounded-xl, rounded-2xl, rounded-top-xl, rounded-b-2xl
