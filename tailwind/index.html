Tailwind CSS

Tailwind CSS is a utility-first CSS framework that lets your build custom designs by composing small, purpose driven classes directly in the HTML

Why do we use tailwind

- there is no need of naming classes
- the output css is very lightweight <10kb
- you can build themes etc using variables
- consistency -> buil-in scales for spacing, coloring, typography

Tailwind properties usage

-> p-4 = gives padding on all sides -> text-center = centers the text -> flex = display:flex -> bg-blue-500 :background-color blue with intensity 500 -> rounded-lg : rounded corners -> shadow:md -> box-shadow:

Tailwind responsive design

mobile-first approach - we start default styles with smaller screens and then add breakpoints prefixes to adjust layouts and styles as the screens get bigger
sm: 640px+
md:768px+
lg:1024px+
xl:1280+
