CSS

Cascading Style Sheets: visual presentation of HTML documents.

used for controlling colors, fonts, spacing, layout design

        css ruleset
        selector{
            property:value
        }
  • selector: targetting HTML element: h1. .card, #header
  • property: the style attributes : color, font-size, margin, font-family
  • value: whatever the property is set to: color:Red, font-size:16px, margin:auto
  • Declaration: property+value: color:Red, line-height:2px
  • Declaration Block: all declaration inside {}

Ways to add CSS

  1. Inline CSS: <h1 style="color:red; font0size:50px; font-family:'Arial'"></h1>
  2. Internal CSS: written inside the style tag
  3. External CSS: a seperate .css file linked to the html
