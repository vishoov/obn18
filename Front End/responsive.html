Responsive Design

Responsive design means building web pages that automatically adapt to their layout, size and appearance on different screens sizes and devices

  • to make our web pages work well on all devices (device compatibility)
  • Improves usability, readability, accessibility (UX)
  • Increases user engagement
  1. viewport meta tag: visible area of a web page, on mobile the width is less, 768px
  2. Fluid Layouts & relative units : elements size themselves accordingly to the size of the viewport or their parent container
  3. Flexible Media : a flexible image always scale down on smaller screens instead of overflowing
  4. MEDIA QUERIES!!!! : let us apply css only when certain conditions are true, maining screen size, orientation etc
  5. Mobile first approach:write default css for small screens, and then use min-width media query
  6. Flexbox and Grid: layout systems designed with responsiveness in mind
