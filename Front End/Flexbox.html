Flex box

Flexible Box Layout is CSS layout model used for distributing space evenly and align items within a container

FlexBox Usage

Flexbox always get implemented on the parent div

  • Main Axis: horizontal axis (by default)
  • Cross Axis: perpendicular to main axis

Flex Direction

defines which direction the main axis runs in, this sets the flow of items to be aligned

  • row-> DEFAULT -> left to right
  • column-> vertically-> top to bottom
  • row-reverse-> same as row, but the order is reversed right -> left
  • column-reverse-> bottom to top

Main Axis Alignment

justify-content helps us distribute the items along the main axis

justify-content has following values

  1. flex-start: DEFAULT-> pack the items on the main axis' start point that is left edge by default
  2. center-> aligns items in the center of the div
  3. flex-end: right align
  4. space-between: equal gaps BETWEEN the items, no gaps at the edges
  5. space-around: equal space around each item including edges
  6. space-evenly: equal gaps everywhere

Cross Axis Alignment

align-items controlls alignment along the cross axis (vertical by default)

  • flex-start:top
  • flex-end: bottom
  • center: vertically in middle
  • stratch (DEFAULT) : items stretch to fill the space
  • baseline: text baseline

align-self: does the same thing as align-items, but applied only to one specific flex item instead of all items, it only overrides align-items setting for the selected element

Flex-Wrap: This is for controlling overflow

  1. no-wrap default
  2. wrap: elements flow into the next line when they run out of available space
  3. wrap-reverse: same as wrap but the order is reversed

Flex Grow, Shrink, Basis

FlexBasis: initial size of the element before any shrinking or growing

flex-grow: defines how much an item grows relative to others when there's extra space

flex-shrink: how much items shrink when there NOT enough space, higher the value more the shrink

Order: decided when an item irrespedtive of its HTML order will be displayed

order is 0 by default any value greater than 0 will push that elmeent to the end, any value less than 0 will bring it towards the start

Align content: this aligns the content as a whole in a certain direction

flex-flow: _flex-direction_ _flex-wrap_
