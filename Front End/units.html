CSS units

CSS has 2 categories of units

Absolute Units Relative Units
Fixed (compared to the size of pixels) Flexible (scale with the content)
Fixed Units Examples

width:200px - pixels is most comment for screens (1920X1080)
width:2cm- centimeters for print layouts

Relative Units Example
.container{
    width:80%; % of parent element's width
    font-size:1.5em; 1.5 times the size of parent's font size;
    font-size:1rem; 1.5 times of the size of 
    ROOT element's font size 
    height:100vh; 100% of viewport height
    width:100vw; 100% of viewport width
}
