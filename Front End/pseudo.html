Pseudo Classes in CSS


            selector:pseudo-class{
            
                property:value
            
            }

Pseudo Class is a keyword added to selector (after a colon) that styles an element based on its state, or position or condition

This gives us a little bit of dynamicity over the styling of our web page


Interactive pseudo classes


  • :hover- when the mouse is over that element
  • :focus - whenever an element has the cursor active in it
  • :active: while the element is activated (pressed down)
  • :visited - already visited links
  • :link - unvisited link

Form UI pseudo classes

  • :valid/:invalid - form fields whenever they pass the form validation or not
  • :required/:optional - whenever required is mentioned in an input
  • :disabled/:enabled :whenever a field is disabled from being filled
  • :checked: for checkboxes/ radio

Functional pseudo classes

  • :not - to exclude any element from the styling
  • :has - has a specific child element or not
  • :is(selector) : matches any element in the selector list (where the inner selector also matches)

Structural Pseudo Classes

  • :first-child - first child of its parent
  • :last-child - last child of its parent
  • :nth-child(n) - nth (n value) child of its parent
  • :nth-last-child(n) - nth (n value) last child of its parent
