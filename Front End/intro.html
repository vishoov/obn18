HTML- Hyper Text Markup Language

Is is a markup language

It uses tags to annotate content and tell the browser what each piece of content IS.

Why this matters

  1. Every single page on the internet is build on HTML
  2. Search engines crawl your HTMLstructure to understand and rank your content (SEO)
  3. browsers only understand HTML for content and structure

The core concepts of HTML -> The HTML TAG!

<tagname attribute="value">content</tagname>

  • Tag: the label that wraps the content (brackets)
  • Attribute: properties of the tag Note: attributes are always written in the opening tag
  • Content: what the element wraps (text, media, link)
  • Final Element: opening tag + content + closing tag
