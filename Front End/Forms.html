HTML Forms

An html form is a structured way of collecting information from the user

  • User registration and login
  • Application forms, feedback forms, escalation forms
  • Chat UIs, Messaging UIs
  • Ecommerce chckout and payment flows

Form Attributes

Attribute Purpose Example
action URL where the form data is being sent action='/register'
method Method using which the data is sent (POST or PUT or PATCH) method="POST"
enctype Enocding Type, required for file uploads enctype='multipart/form-data'
novalidate disable browser's internal validation novalidate
autocomplete Enables/disables browsers autocomplete autocomplete="off"
onSubmit It triggers a javascript function when the form is submitted onSubmit={submitFunc()}

Form Elements

Element Description
input Versatile single line text input used for text, password, email, description, or anything text
label Associates a tag with an input
textarea multi line text input
select dropdown list
button clickable action trigger
fieldset for grouping similar fields together
legend caption for a fieldset
radiomultiple choice

Step by step working flow

  1. broswer reads the form
  2. User fills the fields
  3. User clicks the submit button
  4. browser validates the inputs
  5. Data is serialized
  6. HTTP request will be sent
  7. Server will process the data
  8. Response is returned
