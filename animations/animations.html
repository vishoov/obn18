CSS Animations

let you smoothly chang CSS properties over time using only CSS, these are used for micro interactions in the UI

  • Keyframes : decribe what the changes over time will be (states)
  • animation proerties: decribes how the change will be executed

Keyframe : @keyframe name{rules}

  • to and from 
                @keyframe myAnimation{
                
                from:{opacity:100}
                to:{opactiy:0} 

            }
  • percentage 
                    @keyframe myAnime{
                    
                    0%{
                        opacity:0
                    }
                    100%{
                        opacity:1
                    }

                }
