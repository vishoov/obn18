CSS Animations

let you smoothly chang CSS properties over time using only CSS, these are used for micro interactions in the UI

  • Keyframes : decribe what the changes over time will be (states)
  • animation proerties: decribes how the change will be executed

Keyframe : @keyframe name{states}

  • to and from 
                @keyframe myAnimation{
                
                from:{opacity:100}
                to:{opactiy:0} 

            }
  • percentage 
                    @keyframe myAnime{
                    
                    0%{
                        opacity:0
                    }
                    50%{
                        opacity:0.5
                    }
                    100%{
                        opacity:1
                    }

                }

Animation Properties

  1. animation_name: which is defined using the @keyframe
  2. animation_duration: how lomg the animation cycle will take to complete
  3. animation_timing_function : easing curve (linear, ease, ease-in, ease-out, cubic-bezier(...))
  4. animation-iteration-count: how many times to repeat : 1, 2, 3, 4, infinite
  5. animation-delay: wait before starting of the animation
  6. animation-direction: plaback direction in terms of progress percentage (normal, reverse, alternate, alternate-reverse
  7. animation-play-state: running or paused
  8. animation-fill=-mode: what state to keep after the animation is completed : both, npne, forward, backward 
        animation: name duration timing-function delay iteration_count direction;

The properties that can be animated are transform, opacity, color, background-color, box-shadow, height, width, etc
