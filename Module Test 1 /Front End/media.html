HTML Media Tags

these tags help us EMBEDD media into our web page, including audio, video, interactive text, sections of different products

  • audio
  • video
  • source
  • track
  1. html parser parse the audio tag
  2. src attribute or the child <source> element
  3. checks the MIME type
  4. preload the data based on the attributes present (preload)
  5. renders the controls if the controls attribute is present
  6. fires the javascript events like play, pause, ended, stop
