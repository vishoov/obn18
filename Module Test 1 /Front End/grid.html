Grid

Two dimensional layout system that divides the whole div into smaller chunks with specific locations: 2D Array

2D Array: arr[i][j]

Differences between grid and flex

CSS Grid
FlexBox
> Two Dimensional (rows and columns both)
> One Dimensional (row or column)
> Layout first approach
> Content first approach
> overlapping : true
> overlapping : false

Important terms in grid

  • Grid container
  • Grid items
  • Grid Lines : these are the invisible dividing lines
  • Grid Tracks : rows or column
  • Grid Cell
  • Grid Area

Properties

Layout properties

  1. grid-template-columns: number of columns with their values as their widths
  2. grid-template-rows: number of rows with their values as their heights
  3. grid-template:rows/columns

Content Properties

  1. grid-column-start: where an element starts in the grid (column wise)
  2. grid-column-end: where the element ends column wise
  3. grid-row-start: where an element starts in the grid (row wise)
  4. grid-row-end
  5. grid-column: start_column_line / end_column_line shorthand for column start and column end
  6. grid-row: start_row_line / end_row_line shorthand for row start and row end
  7. grid-area: shorthand for grid_row / grid_column

Alignment Properties

  • justify-content: aligns the grid tracks equally in the grid
  • align-content: arrange the contend vertically
  • place-content : align / justify
