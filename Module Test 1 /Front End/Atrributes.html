Attributes HTML

Attributes are specifications placed inside the html element's opening tag, that adds additional information about that element, hooks the styles it to , and helps build the behaviour

Type Description Example
Global Attributes Attributes applicable on all html elements class, id, title, style, hidden, tabindex
Element Specific those attributes that belong to specific tags only href, src, required, action, target, onSubmit
Boolean Attributes presence = true, absence= false autoplay, checked, disabled, required
Custom Attribute (data-*) store product specific data data-user-id="42"

Alt Tag: whenever an image doesnt load, the alt text appears
