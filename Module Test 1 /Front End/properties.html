Display Properties

display property controls how an element participates in the layout
  • BLOCK: takes full width, stacks vertically
  • INLINE: only as wide as the content, flows with the text
  • INLINE-BLOCK: inline flow but accepts width/height
  • NONE: hidden element


Position Property

The position property REMOVES the element from the normal document flow

  • STATIC: this is the default, follows the normal document flow
  • RELATIVE: offset from normal position, space is also preserved
  • ABSOLUTE: removed from the flow, positioned relative to the nearest non-static element element or the parent NOTE: ALWAYS USE RELATIVE IN PARENT TO USE ABSOLUTE IN CHILD
  • FIXED: it fixes an element to the viewport
  • STICKY: relative untill scroll threshold, then fixed
